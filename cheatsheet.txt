Camped UI Cheatsheet with Tags and Usage

=========================================


Layout & Display
----------------
- @camped-ui/accordion
  Description: Expandable/collapsible content sections used for FAQs and disclosures.
  • <Accordion>
  • <AccordionContent>
  • <AccordionItem>
  • <AccordionTrigger>

- @camped-ui/alert
  Description: Used to show important messages or warnings.
  • <Alert>
  • <AlertDescription>
  • <AlertTitle>

- @camped-ui/aspect-ratio
  Description: A loading placeholder used while content is being fetched or rendered.
  • <AspectRatio>

- @camped-ui/avatar
  Description: User profile image with fallback initials.
  • <Avatar>
  • <AvatarFallback>
  • <AvatarImage>

- @camped-ui/badge
  Description: Indicates status, notifications, or achievements. It enhances visual communication within user interface.
  • <Badge>

- @camped-ui/bento-grid  
  Description: Responsive skewed grid layout for organizing and showcasing content.  
  • <BentoGrid>  
  • <BentoGridItem>

- @camped-ui/breadcrumb  
  Description: Displays the path to the current resource using a hierarchy of links.  
  • <Breadcrumb>  
  • <BreadcrumbItem>  
  • <BreadcrumbLink>  
  • <BreadcrumbList>  
  • <BreadcrumbPage>  
  • <BreadcrumbSeparator>

- @camped-ui/button  
  Description: Interactive element used to perform actions, submit forms, or navigate.  
  • <Button>  

- @camped-ui/card  
  Description: Structured container for presenting content like articles, product details, or user profiles.  
  • <Card>  
  • <CardContent>  
  • <CardDescription>  
  • <CardFooter>  
  • <CardHeader>  
  • <CardTitle>  

- @camped-ui/carousel

  • <Carousel>  
  • <CarouselContent>  
  • <CarouselItem>  
  • <CarouselNext>  
  • <CarouselPrevious>

  • <CommandDialog>  
  • <CommandEmpty>  
  • <CommandGroup>  
  • <CommandInput>  
  • <CommandItem>  
  • <CommandList>  
  • <CommandSeparator>

  • <ContextMenu>  
  • <ContextMenuContent>  
  • <ContextMenuItem>  
  • <ContextMenuTrigger>

  Description: Feature-rich table and datagrid built with TanStack Table, supporting sorting, filtering, and large datasets.  
  • <Table>

  • <DropdownMenuContent>

  • <DropdownMenuLabel>  
  • <DropdownMenuSeparator>  
  • <DropdownMenuTrigger>

  • <HoverCardContent>  
  • <HoverCardTrigger>  

- @camped-ui/label  
  Description: Accessible labels associated with form controls to improve usability and inclusivity.
